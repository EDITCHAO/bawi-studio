🚀 PROCHAINES ÉTAPES:

1. Installation des dépendances:
   Windows: install.bat
   Linux/Mac: ./install.sh

   Terminal 1: cd frontend && npm run dev
   Terminal 2: cd backend && npm run dev

4. Vérification:
