## 🔍 Monitoring

### Logs

#### Render (Backend)
1. Aller sur https://render.com
2. Sélectionner le service
3. Aller dans Logs
4. Vérifier les erreurs

#### Vercel (Frontend)
1. Aller sur https://vercel.com
2. Sélectionner le projet
3. Aller dans Deployments
4. Voir les logs

## 🔄 Mises à jour

### Vérifier les mises à jour

```bash
# Frontend
npm outdated

# Backend
cd backend
npm outdated
```

### Mettre à jour les dépendances

```bash
# Frontend
npm update

# Backend
cd backend
npm update
```

## 💾 Backups

### Backup Supabase

1. Aller sur https://supabase.com
2. Sélectionner le projet
3. Aller dans Backups
4. Vérifier les backups automatiques
