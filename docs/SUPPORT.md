# Support - Bawi Studio

## 📞 Canaux de support

### 🐛 Signaler un bug

1. **GitHub Issues**: https://github.com/bawi-studio/bawi-studio/issues
2. **Email**: support@bawi-studio.com

### 💡 Demander une fonctionnalité

1. **GitHub Discussions**: https://github.com/bawi-studio/bawi-studio/discussions
2. **Email**: features@bawi-studio.com

### ❓ Poser une question

1. **GitHub Discussions**: https://github.com/bawi-studio/bawi-studio/discussions
2. **Email**: help@bawi-studio.com

### 🔐 Signaler une vulnérabilité

1. **Email**: security@bawi-studio.com

**Ne pas publier publiquement!**

## 📋 Avant de contacter le support

- [ ] Vérifier les logs du navigateur (F12)

### Vérifier les issues existantes

- [ ] Chercher sur Google
